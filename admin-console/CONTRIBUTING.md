# Contributing to Admin Console

Thank you for considering contributing to the Admin Console project! We welcome contributions from everyone.

## How to Contribute
1. **Fork the repository** to your own GitHub account.
2. **Clone your fork** locally.
   ```bash
   git clone https://github.com/your-username/admin-console.git
   cd admin-console
   ```
3. **Create a new branch** for your feature or bug fix.
   ```bash
   git checkout -b feature/your-feature
   ```
4. **Make your changes** and ensure to add tests where applicable.
5. **Commit your changes** with a clear message about what you have done.
   ```bash
   git commit -m "Add feature XYZ"
   ```
6. **Push to your fork** on GitHub.
   ```bash
   git push origin feature/your-feature
   ```
7. **Open a Pull Request** to the main repository.

## Code of Conduct
Please make sure to read our [Code of Conduct](./CODE_OF_CONDUCT.md) before contributing.

## Issues
If you find a bug or have a feature request, please open an issue on GitHub.
