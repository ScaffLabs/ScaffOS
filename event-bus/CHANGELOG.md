# Changelog

## [1.0.0] - 2023-10-01
- Initial release of the Event Bus service with core functionality including event creation, retrieval, updating, and deletion.
- Implemented Redis for message brokering.
- Added in-memory storage for events.
- Included validation with Zod and error handling middleware.
- Rate limiting implemented for API endpoints.

## [1.0.1] - 2023-10-02
- Added health check endpoint to monitor service health.
- Updated documentation with API reference and environment variables.

## [1.0.2] - 2023-10-03
- Added detailed README.md with comprehensive setup instructions, API reference, and environment variables table.
- Updated CHANGELOG.md for version history.

## [1.0.3] - 2023-10-04
- Enhanced error handling with more descriptive messages.
- Improved API documentation for clarity and completeness.
