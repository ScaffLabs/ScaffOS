# Changelog

All notable changes to this project will be documented in this file.

## [1.3.0] - 2023-12-15
- Comprehensive documentation added, including README and CHANGELOG updates.
- Improved error handling in the order API.

## [1.2.0] - 2023-12-01
- Introduced a new order submission feature.
- Added tests for critical functionalities.

- Fixed minor bugs in position management.
- Updated dependencies to address security vulnerabilities.

## [1.1.0] - 2023-11-01
- Added enhanced error handling for API requests.
- Improved documentation for the API endpoints.

## [1.0.0] - 2023-10-01
- Initial release with core features:
  - User can create, update, and delete positions.
  - Real-time charting for price data.
  - Integration with external data API.
