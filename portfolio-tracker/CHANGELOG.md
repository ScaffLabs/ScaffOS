# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2023-11-20
### Added
- Initial project setup with portfolio tracking functionality.
- Comprehensive API for managing portfolios.
- Detailed README.md with project description, setup instructions, and API reference for every endpoint.
- Inline comments in the codebase explaining complex logic for better understanding.
- Comprehensive CONTRIBUTING.md with clear guidelines on contributing to the project.

### Changed
- Updated existing API responses to include more detailed error messages.
- Improved formatting and readability in README.md and CHANGELOG.md.

## [1.1.0] - 2023-12-01
### Added
- Health check and readiness endpoints for service monitoring.
- Rate limiting middleware to prevent abuse of the API.

### Changed
- Improved error handling and logging throughout the application.
- Enhanced validation for portfolio creation and update requests.
